弧度：弧长等于半径的弧，其所对的圆心角为1弧度。(即两条射线从圆心向圆周射出，形成一个夹角和夹角正对的一段弧。当这段弧长正好等于圆的半径时，两条射线的夹角的弧度为1)。 圆的周长： 2πr 一周的弧度： 2πr / r = 2π 一周 360° 所以 360° = 2π; 即 一角度 等于 π/180 弧度 1° = π/180; 一弧度 等于 180/π 角度 1rad = 180/π； 求鼠标跟坐标系原点的连线与x轴的夹角 使用 Math.atan2(); 而不是Math.atan(); Math.atan 与X轴的夹角， 一三象限为负 范围为 0 ~ 90度 二四象限为正 范围为 0 ~ -90度 无法正确判断距离X轴的夹角 Math.atan2 X轴上方 0 ~ -180度 X轴下方 0 ~ 180度 可以通过 atan2 确定与Xz轴的夹角
